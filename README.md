# FlightGear Mirage 2000 HDR Resources

Resources (mostly 3D) for the FlightGear Migrage 2000 model at https://github.com/5H1N0B11/flightgear-mirage2000. There is a similar repo at https://github.com/5H1N0B11/flightgear-Mirage2000-Ressources. The difference is that this repo is for the HDR/next version of FlightGear, while the other repo is for the current and older versions of FlightGear: 2024.1.
